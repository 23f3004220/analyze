Welcome

This project demonstrates a simple data processing workflow using Python, Pandas, and GitHub Actions. It involves reading data from an Excel file, converting it to CSV, processing it, and publishing the results.

Explore the README.md for detailed setup and execution instructions.

Core Script: execute.py

The execute.py script processes tabular data. It was updated to fix a non-trivial error related to data type handling, ensuring robust aggregation.

Data Files

Data is initially provided in data.xlsx. This file is converted to data.csv as part of the workflow, which execute.py then reads.

Continuous Integration & Deployment

A GitHub Actions workflow (.github/workflows/ci.yml) automates linting with Ruff, executes the data processing script, and publishes the resulting result.json to GitHub Pages.

Check the GitHub repository's Actions tab for CI/CD logs and the GitHub Pages deployment.
